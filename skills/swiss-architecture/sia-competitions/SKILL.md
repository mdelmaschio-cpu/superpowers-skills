# /sia-competitions — Swiss Architecture Competitions

## SIA 142 vs SIA 143

| | **SIA 142** | **SIA 143** |
|---|---|---|
| Type | Architecture competition (Wettbewerb) | Study commission (Studienauftrag) |
| Anonymity | **Anonymous** — jury does not know authors | **Not anonymous** — teams known |
| Participants | Open or selected | Selected (typically 3–6 teams) |
| Compensation | Prize money (Preissumme) only | **Fee paid to all participants** |
| Outcome | Binding: winner receives mandate (unless reasons not to) | Advisory: jury recommends, client retains discretion |
| Best for | Public buildings, sensitive sites, high visibility | Complex briefs, experimental approaches, tight budgets |

## Competition Types (SIA 142)

| Type | Description |
|------|-------------|
| **Offener Wettbewerb** | Open to all qualified architects |
| **Selektiver Wettbewerb** | Pre-selected participants (typically 5–20) |
| **Einladungswettbewerb** | Invitation competition (3–10 selected firms) |
| **Ideenwettbewerb** | Ideas competition — no mandate guaranteed |
| **Projektwettbewerb** | Project competition — winner receives mandate |
| **Einstufig** | Single stage |
| **Zweistufig** | Two-stage: concept → detailed project |

## Submission Requirements (Typical)

Competitions specify their own requirements in the programme. Typical elements:

**Plans:**
- Site plan 1:500 or 1:1000
- Floor plans at 1:200 (all floors)
- Key sections and elevations at 1:200
- Façade detail(s) at 1:50 or 1:20

**Representations:**
- Exterior visualization (axonometric, perspective, or model photo)
- Interior visualization(s)
- Concept diagram / parti

**Text:**
- Project description (Erläuterungsbericht): concept, structure, materials, energy, accessibility
- Word/character limits specified in programme

**Technical:**
- Area calculations per SIA 416
- Rough cost estimate per BKP
- Energy concept summary

**Model:**
- Site model at 1:500 or 1:200 (often required, sometimes optional)

## Anonymity Rules (SIA 142)

- No identifying marks on any submission material
- Submission plan code assigned by organizer
- All communications through secure portal
- **Disqualification** if identity can be determined from submission

## Jury (Preisgericht)

- Typically 5–7 members: Fachpreisrichter (professionals) + Sachpreisrichter (client representatives)
- Independent jury chair
- Jury decision is binding for Projektwettbewerb (client must award to winner or explain why not)
- Jury report (Bericht des Preisgerichts) published and distributed to all participants

## Study Commission (SIA 143) Process

1. Client selects 3–6 teams (qualification required)
2. Briefing meeting with all teams (Startgespräch)
3. Intermediate review (Zwischenkolloquium) — feedback round
4. Final presentations to jury
5. Jury recommendation (not binding)
6. Client awards mandate (typically to recommended team)

**Compensation per team**: typically CHF 15'000–50'000 depending on project scale

## IVöB / WTO — Public Procurement Link

For public clients (cantons, municipalities):
- Projects above thresholds **must** use SIA 142/143 or equivalent competition procedure
- **IVöB threshold** for design mandates: CHF 150'000 (open procedure), CHF 250'000 (international WTO threshold)
- Competitions count as compliant procurement procedure

## Practical Tips

- Read the Programm (competition brief) three times before designing
- Note exact submission format requirements — disqualifications happen for format violations
- Mark your model with submission number, not name
- Submit via simap.ch (public competitions) or client's specified platform
- **After announcement**: jury report is valuable feedback regardless of result
