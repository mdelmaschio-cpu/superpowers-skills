# /sia-phases — SIA 102 Project Phases

Structured breakdown of Swiss architecture project phases per SIA 102, with deliverables, milestones, and typical fee distribution.

## Phase Overview

| Phase | Code | Typical Fee % | Client Decision |
|-------|------|--------------|-----------------|
| Strategic Planning | SP | 1–3% | Go/no-go |
| Preliminary Studies | VS | 5–10% | Concept approval |
| Preliminary Project | VP | 10–15% | Building permit submission |
| Building Project | BP | 20–25% | Tender approval |
| Tendering | AS | 5–8% | Contract award |
| Construction Management | RL | 30–40% | Handover |
| Post-construction | — | 2–3% | Final account |

*Percentages vary by project category and complexity. Always refer to SIA 102 Appendix C.*

## Phase Deliverables

### Vorstudien (VS) — Preliminary Studies
- Site analysis and constraints
- Feasibility study
- Program verification
- Reference projects
- Rough cost estimate (±30%)
- **Client decision: confirm program and budget**

### Vorprojekt (VP) — Preliminary Project
- Architectural concept
- Floor plans, sections, elevations (1:200)
- Structural and MEP concepts
- Building permit pre-consultation
- Cost estimate ±15% (BKP)
- **Client decision: approve for permit submission**

### Bauprojekt (BP) — Building Project
- Detailed plans (1:50)
- All technical specifications
- Material selections
- Cost estimate ±10%
- Tender documents prepared
- **Client decision: approve tender launch**

### Ausschreibung (AS) — Tendering
- Leistungsverzeichnis (LV) — bill of quantities
- Submission documents per IVöB/LMP (if public)
- Bid evaluation and recommendation
- Contract negotiation support
- **Client decision: award contract**

### Realisierung (RL) — Construction Management (Bauleitung)
- Construction supervision
- Bauleiterprotokoll (weekly site meetings)
- Work inspection and approval
- Change order management (Nachtragsmanagement)
- Cost controlling vs. BKP
- Abnahme (formal handover with defect list)
- **Client decision: accept building**

### Bewirtschaftung (BW) — Post-construction
- As-built documentation
- Defect tracking (Mängelrüge)
- Final cost accounting
- MINERGIE or SIA 380 verification if applicable

## Key Documents per Phase

| Document | Phase | Purpose |
|----------|-------|---------|
| Raumprogramm | VS | Space program agreed with client |
| Baubewilligungsgesuch | VP | Building permit application |
| Submission/LV | AS | Tender bill of quantities |
| Werkvertrag | AS | Construction contract (SIA 118) |
| Bauleiterprotokoll | RL | Site meeting minutes |
| Abnahmeprotokoll | RL | Handover inspection record |
| Schlussabrechnung | post-RL | Final cost account |

## BKP Cost Categories (Baukostenplan)

| BKP | Category |
|-----|----------|
| 0 | Grundstück (Land) |
| 1 | Vorbereitungsarbeiten (Site preparation) |
| **2** | **Gebäude (Building) — basis for SIA 102 fees** |
| 3 | Betriebseinrichtungen (Equipment) |
| 4 | Umgebung (Landscaping) |
| 5 | Baunebenkosten (Ancillary costs) |
| 9 | Ausstattung (Furnishings) |
