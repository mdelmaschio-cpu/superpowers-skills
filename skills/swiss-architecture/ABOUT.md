# Swiss Architecture Skills — Attribution

Skills written from scratch for architects and construction managers operating in Switzerland. Based on:

- **SIA norms** (Schweizerischer Ingenieur- und Architektenverein / Société des ingénieurs et des architectes)
- **Swiss federal and cantonal building law**
- **MINERGIE** certification standards
- **Swiss public procurement law** (IVöB/LMP)

These skills have no direct upstream project — authored for mdelmaschio-cpu/superpowers-skills.

## Skills Included

- `sia-norms` — Reference guide to the SIA norm system
- `sia-phases` — Project phases per SIA 102
- `building-permit-ch` — Cantonal building permit process
- `bkp-cost-plan` — BKP building cost planning per SIA 416
- `minergie-energy` — MINERGIE and Swiss energy performance
- `bauleitung-ch` — Construction site management documentation
- `sia-118-contract` — Construction contract analysis per SIA 118
- `cantonal-zoning-ch` — Swiss cantonal zoning and land use
- `sia-competitions` — Architecture competitions per SIA 142/143
- `accessibility-ch` — Swiss accessibility norms (SIA 500, nLHin)
